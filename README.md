# JAIT (propal)

## Download projects

```bash
git clone https://github.com/JAIT-blog/JAIT
```

## Setup develloppment environnment quickly

```
./quick_setup.sh
```
## Run Flask server

```bash
source bin/activate
flask run # or python3 -m flask run
```

To leave virtual env, tap :

```bash
deactivate
```
